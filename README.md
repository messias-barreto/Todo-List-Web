# Todo-List-Web
Projeto com o foco de aprofundar os estudos em React.Js

## Sobre o Projeto
Sistema criado com a finalidade de gerenciar a Lista de Tarefas de Diversos Projetos 

## Funcionalidades
- criar conta 
- criar / editar projeto 
- adicionar / remover as tarefas no projeto
